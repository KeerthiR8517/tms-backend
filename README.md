## Important Note

The backend server is deployed on Render free tier. 
If the server has been idle, the first request may take a few minutes to start. 
Once it starts, all features including login and data display work normally.

This is due to hosting limitations and is expected behavior.

- **Pagination:** Pagination in the shipments list starts after 6 entries. 
  Additional entries will appear on the next page.
